Welcome to the PenguinSurf Kodi Repository

The official source for the PenguinSurf suite of Kodi Addons, providing region-free access to a wide range of content including Movies, TV Shows, Live TV, Sports, and Documentaries.

Installation Instructions

Follow these steps to install the PenguinSurf Repository in Kodi:

  1. Open **Kodi**.
  2. Go to **Settings** (the gear icon) -> **File Manager** -> **Add source**.
  3. In the text box labeled <None>, enter the repository URL:
    https://hadimariaali-droid.github.io/PenguinSurf/
  4. Name the source PenguinSurfRepo and click **OK**.
  5. Go back to the main menu and select **Add-ons**.
  6. Select **Install from zip file** (you may need to enable unknown sources).
  7. Select the source you just added: PenguinSurfRepo.
  8. Select the repository file: repository.penguinsurf.zip.
  9. Once installed, select **Install from repository**.
  10. Choose **PenguinSurf Repository** and install the addons you want!

Addons Included in the Repository

  • PenguinSurf (Video Addon): Movies and TV Shows with multi-version support (Kodi 19, 20, 21).
  • ArcticLive (Video Addon): All English, Arabic, Japanese, Korean, French, Spanish, German, Italian live TV channels.
  • SPEN (Sports Penguin Entertainment Network): All English, Arabic, Japanese, Korean, French sports channels.
  • PenguinDocumentary (Video Addon): All English, Arabic, French, Japanese, Korean documentary movies.
  • ScrapePenguin (Module): Core scraping and conceptual region-free logic module.
  • PenguinSurf Setup Wizard (Program Addon): Helps configure and install the full suite of addons.

Repository Maintenance

For developers, the repository files are hosted directly in this GitHub repository. The generate_repo.py script can be used to update the addons.xml and addons.xml.md5 files after any changes.
